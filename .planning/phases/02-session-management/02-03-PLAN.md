**File 1: `src/lib/actions/sessions.ts`** — Add `editSession` function to existing file.

```typescript
export async function editSession(
  sessionId: string,
  scope: 'this' | 'future',
  formData: FormData
): Promise<SessionActionResult> {
```

Flow (from RESEARCH Pattern 2):
1. Auth check: getUser(), role must be coach or admin
2. Parse formData with `EditSessionSchema.safeParse()` — only include fields that were actually changed (non-empty values)
3. Build updates object from parsed data, converting `start_time` to full `scheduled_at` if provided (combine with current session date)

**scope === 'this':**
4a. Update the `sessions` row directly: `supabase.from('sessions').update(updates).eq('id', sessionId)`

**scope === 'future':**
4b. Get session's `template_id` and `scheduled_at`: `supabase.from('sessions').select('template_id, scheduled_at').eq('id', sessionId).single()`
5b. If no `template_id`, return `{ success: false, error: 'Cannot edit future sessions — this session has no template' }`
6b. Build template-level updates (venue, capacity, start_time, duration_minutes — not court_number which is per-instance)
7b. Update template: `supabase.from('session_templates').update(templateUpdates).eq('id', session.template_id)`
8b. Update all future sessions: `supabase.from('sessions').update(updates).eq('template_id', session.template_id).gte('scheduled_at', session.scheduled_at)` — CRITICAL: `gte` not `gt` to include the current session
9b. Do NOT update past sessions (the `gte` handles this)

10. `revalidatePath('/sessions')` and `revalidatePath('/coach')`
11. Return `{ success: true }`

```typescript
import { createClient } from '@/lib/supabase/server'
import { notFound } from 'next/navigation'
import { EditSessionForm } from '@/components/sessions/EditSessionForm'

export default async function EditSessionPage({ params }: { params: Promise<{ sessionId: string }> }) {
  const { sessionId } = await params
  const supabase = await createClient()
  const { data: session } = await supabase.from('sessions').select('*').eq('id', sessionId).single()
  if (!session) notFound()
  return (
    <div className="min-h-screen bg-background">
      <div className="max-w-[560px] mx-auto px-4 py-8">
        <h1 className="font-display text-[28px] font-bold text-foreground mb-6">Edit session</h1>
        <EditSessionForm session={session} templateId={session.template_id} />
      </div>
    </div>
  )
}
```

Note: In Next.js 16, `params` is a Promise and must be awaited.

# Phase 4 — UI Design Contract: Coach Dashboard & Community Events

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.
> **Design source:** AceHub prototype — fresh, sporty, young. Green primary + warm yellow-orange secondary.
> **Continuity:** All tokens, spacing, typography, and component patterns are carried forward from the Phase 3 UI-SPEC unchanged. This document extends Phase 3 with new surfaces only.

## Spacing Scale

Identical to Phase 3. All spacing uses the 8-point scale. Do not use arbitrary values unless listed as an exception.

| Token | Value | Tailwind | Usage |
|-------|-------|----------|-------|
| xs | 4px | `gap-1`, `p-1` | Badge padding, icon gaps |
| sm | 8px | `gap-2`, `p-2` | Inline element spacing |
| md | 16px | `gap-4`, `p-4` | Default card padding, field spacing |
| lg | 24px | `gap-6`, `p-6` | Section padding |
| xl | 32px | `gap-8`, `p-8` | Page section breaks |
| 2xl | 48px | `py-12` | Major layout gaps |
| 3xl | 64px | `py-16` | Page top padding on desktop |

Exceptions:
- Page horizontal padding: 20px — `px-5` (AceHub pattern, Phase 3 established)
- Bottom nav height: 64px fixed — `pb-16` plus `pb-[env(safe-area-inset-bottom)]` for iOS
- Touch targets: 44px minimum on all tappable elements — `min-h-[44px]`
- Input height: 48px — `h-12`
- Attendee avatar strip circles: 24px — `w-6 h-6 rounded-full`; overlap with `−ml-1`
- Day/week toggle pill height: 32px — `h-8`
- Event type selector buttons (step 1): 64px — `h-16 rounded-2xl`

## Typography

Identical to Phase 3. No new sizes or weights introduced.

| Role | Font | Size | Weight | Line Height | Tailwind |
|------|------|------|--------|-------------|----------|
| Display heading | Space Grotesk | 24px | 700 | 1.2 | `font-heading font-bold text-2xl leading-tight` |
| Stat value | Space Grotesk | 24px | 700 | 1.2 | `font-heading font-bold text-2xl` |
| Section heading / card title | Space Grotesk | 16px | 700 | 1.5 | `font-heading font-bold text-base` |
| Body / label / helper | Inter | 14px | 400 | 1.5 | `text-sm` |
| Micro label | Inter | 10px | 400 | 1.4 | `text-[10px]` |

Rules (carried from Phase 3):
- Exactly 4 font sizes: 24px, 16px, 14px, 10px. Do not introduce any other size.
- Exactly 2 font weights: 400 (Inter body/labels) and 700 (Space Grotesk headings). Do not use 500 or 600.
- All headings use `font-heading font-bold` regardless of size.
- Body text, labels, helper text, badge text: default `font-sans` (Inter 400).
- Stat labels below values: `text-[10px] text-muted-foreground uppercase tracking-wide`.
- Bottom nav labels: `text-[10px]`, `font-bold` when active.
- "See all" section links: `text-sm text-primary`.
- Event card date/venue meta: `text-sm text-muted-foreground`.
- Announcement body text: `text-sm text-foreground`.

- All form inputs: `<Label htmlFor>` paired with input `id`

- Date navigation chevrons: `aria-label="Previous day"` / `aria-label="Next day"`
- FAB: `aria-label="Create event"`
- Edit event button: `aria-label="Edit event"`
- Edit announcement button: `aria-label="Edit announcement"`
- Attendee avatar strip: container `aria-label="{N} attendees" role="img"`; individual avatars `aria-hidden="true"`
